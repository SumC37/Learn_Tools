# 越界修改对象

- 特别通道：本质为在g()中直接修改`global`中的`copyright`
```
def g():
	global copyright
```

- 夹层区域`enclosing`：`nonlocal`非当前区域
```
def g():
	nonlocal copyright
```
	
- `nonlocal copyright`会与离它最近的重写过`copyright`的层建立特别通道

- 列表这类自带增删功能的对象可跨封装界限增删

# python的设计格言

`import this`

```
a = 5
b = 5
c = a
```

- c语言：给`a,b,c`各自单独创建对象`5`
- python：`a,b`后为具体对象，各*新建对象*；`c`后为变量名称，与`a`*共享对象*
- python只新建不修改（列表除外）
