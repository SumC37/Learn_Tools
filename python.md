#fleeting 
信息直接来源：deepseek

1. **没有块级作用域**：
    - Python **没有**像 Java 或 C++ 那样的块级作用域（花括号 `{}` 定义的作用域）
    - `with`、`if`、`for` 等语句块**不会**创建新的作用域
    - 在代码块内部定义的变量，在块外部仍然可见
        
2. **函数作用域**：
    - 只有**函数**会创建新的作用域
    - 在函数内部定义的变量，在函数外部不可见（除非使用 `global` 或 `nonlocal` 声明）
        
3. **模块作用域**：
    - 在模块级别（即不在任何函数内）定义的变量是全局变量

4. **变量生命周期**：
    - `with` 块结束后，文件会自动关闭，但变量仍然存在
    - 变量绑定的数据（列表）仍然在内存中
        
5. **文件资源管理**：
    - `with` 语句只负责管理文件资源（确保文件关闭）
    - 它不影响已经加载到内存中的数据

- 任何整数与1进行按位与，实际上是在检查其最低位（二进制表示的最右边一位）。如果最低位是0，则结果为0，说明是偶数；如果最低位是1，则结果为1，说明是奇数。
