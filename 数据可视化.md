# 直角坐标系

- 第三方库`matplotlib plotly pandas`
```python
import matplotlib.pyplot as plt

x = ['01','02','03','04','05','06']
y1 = [2,3,1,3,2,3]
y2 = [6,7,7,9,8,7]

plt.style.use('ggplot')
fig,ax = plt.subplots() #创建直角坐标系

ax.plot(x,y1,c='red') #只有一个参数时该参数为y值
ax.plot(x,y2,c='blue')
ax.fill_between(x,y1,y2,facecolor='green',alpha=0.1) #在两条折线间填充颜色

ax.axes.xaxis.set_ticks(range(0,60,10)) #只显示横轴上选定的索引值，也可[]
ax.axes.xaxis.set_ticklabels([1,2,3,4,5,6]) #给定值
#ax.axes.xaxis.set_ticks(range(0,60,10),[1,2,3,4,5,6])
#plt.xticks(range(0,60,10),[1,2,3,4,5,6])
fig.autofmt_xdate() #把横轴的值斜过来

plt.rcParams['font.sans-serif'] = ['Microsoft YaHei']
ax.set_title('测试图',fontsize=18)
ax.set_xlabel('x',fontsize=14)
ax.set_ylabel('y',fontsize=14)
ax.tick_params(axis='both',which='major',labelsize=14)

plt.show() #显示
```

- `plot()`折线图  `scatter`散点图
